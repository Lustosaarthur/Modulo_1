Tags em HTML

Em HTML 5 temos algumas tags muito importantes, veja algumas abaixo

  • Paragrafos
    • para paragrafos utilizamos a tag (P)

  • Titulos
    • Para titulos podemos utilizar desde a taga (H1) até a (H6)
  • Imagem
    • Para imagens utilizamos a tag (IMG)
  • Abreviações
    • Para abreviações utilizamos a tag (abbr)
  • Codigo fonte
    • Para codigo fonte usamos a taga (code)
  • Listas numeradas
    • Para listas numeradas usamos a tag (OD - ordered list) com varios parametros permitidos
  • Listas com demarcadores
    • para usarmos listas com demarcadores podemos utilizar a tag (dt) e a tag (dd) para demarcara o
